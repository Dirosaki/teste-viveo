
# Teste Viveo | Front-End


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Dirosaki/teste-viveo.git
```

Entre no diretório do projeto

```bash
  cd teste-vivveo
```

Instale as dependências

```bash
  yarn install
```

Inicie o servidor

```bash
  yarn dev
```
